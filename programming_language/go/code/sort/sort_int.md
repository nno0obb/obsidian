# # TSIA

```go
// main.go
package main

import (
	"fmt"
	"sort"
)

func main() {
	slice := []int{5, 1, 4, 2, 3}
	sort.Ints(slice)
	fmt.Println(slice)
}
```

```shell
$ go run main.go
[1 2 3 4 5]
```
